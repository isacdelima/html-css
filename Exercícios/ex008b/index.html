More some tags

Código-fonte

O código-fonte consiste simplesmente no código escrito na linguagem de programação utilizada pelo programador.

Para mostrar um código no html se utiliza da semântica usando a tag <code>
usando essa tag o código fica assim:

if number1 + number2 = 6 {return'O resultado é 6'} else {return 'O número não é 6'}

Pré-formatação

A pré-formatação é utilizada para mostrar o texto exatamente da forma que ele foi escrito, incluindo espaçamentos,posições,etc. se usa a tag <pre>

            
        if number1 + number2 = 6
        {return'O resultado é 6'}
        else {return 'O número não é 6'}

Quotation (Citações)

Simple quotation

A citação simples consiste em uma citação onde não há quebra de linha. Se utiliza a tag <q>

Deus, para a felicidade do homem, inventou a fé e o amor

Complete quotation

A citação completa é uma citação com quebra de linha, se usa a tag <blockquote>

Doesn't everyone sell his soul? I tell you, sir: the devil does not exist, there is no devil, yet I sold him my soul. That is what I am afraid of. To whom did I sell it? That is what I am afraid of, my dear sir: we sell our souls, only there is no buyer.
