Formatação de texto básica

Negrito / Destaque

Nesta frase temos um texto em negrito usando a tag 'b' (Não semântico)

Já nesta frase temos um texto em destaque usando a tag 'strong' (Semântica)

Itálico / Ênfase

Nesta frase temos um texto em itálico utilizando a tag 'i' (Não semântica)

Nesta frase temos um texto em ênfase utilizando a tag 'em' (Semântica)

Marcação

Para deixar o texto marcado se utiliza a tag 'mark'

Para mudar a cor do marcador pode se adicionar um style dentro da tag mark ou adicionando um seletor mark no style do head

Texto grande e pequeno

Exemplos : big small

Atualmente a tag 'big' está obsoleta por ser não semântica, já a tag 'small' possui semântica pois é utilizada como 'letras miúdas' de um contrato,avisos,regras,etc...

Texto inserido

Podemos marcar como texto inserido quando queremos enfatizar que esse texto foi inserido depois, usando a Tag 'ins'

Existe também o 'sublinhado' denotado pela tag 'u' (Não semântico)

Texto deleteado

O texto pode ser 'riscado' pra enfatizar de que ele foi desconsiderado ou deletado usando a tag 'del'

Texto sobrescrito

O texto sobrescrito é aquele que fica acima do texto, por exemplo : x2+30 para isto se usa a tag 'sup'

Texto subscrito

O texto subscrito é aquele que fica abaixo do texto, por exemplo : H2O para isto se usa a tag 'sub'
